# 🏥 MediScan — Emergency Medical QR System

> Because in an emergency — every second counts.

## 🚨 Problem
Every day thousands of accidents happen. When a victim is unconscious,
rescuers don't know their blood group, allergies, or who to call.
That costs lives.

## ✅ Solution
MediScan lets anyone register their medical details and get a unique QR code.
Print it. Stick it on your helmet. If an accident happens — rescuer scans
the QR with any phone and instantly sees all emergency information.

## ⚡ Features
- Register medical profile in under 1 minute
- Generate unique QR code instantly
- Scan QR with any phone camera — no app needed
- View blood group, allergies, conditions instantly
- One tap CALL NOW to reach emergency contact
- Works on local WiFi — no internet needed
- Webcam scanner for hospital/medical team use

## 🛠 Tech Stack
- Python
- Streamlit
- SQLite
- OpenCV
- qrcode library

## 🚀 How to Run

### 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/mediscan-emergency-qr.git
cd mediscan-emergency-qr

### 2. Install dependencies
pip install -r requirements.txt

### 3. Run the app
streamlit run app.py

### 4. Open on phone
Check terminal for Network URL like:
http://192.168.x.x:8501
Open this on your phone (same WiFi)

## 📱 How it Works
1. Register on MediScan — fill name, blood group, allergies, conditions, contact
2. Download QR code — print and stick on helmet or wallet
3. Accident happens — rescuer scans QR with phone
4. Emergency info appears instantly on screen
5. Tap CALL NOW — family gets the call
